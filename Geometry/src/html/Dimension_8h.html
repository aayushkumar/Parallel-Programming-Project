Detailed Description

Role: Functions for converting from 1D indices to 3D co-ordinate numbers, and back again.

Assumptions:

Comments: These macros should be used anywhere where IJK-style conversions are required. It was created since the FeEquationNumber needed to use new co-ordinate systems.
